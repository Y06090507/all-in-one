# OOP 面向对象设计

## SOLID 原则
- [ ] **S** — 单一职责 (Single Responsibility)
- [ ] **O** — 开闭原则 (Open/Closed)
- [ ] **L** — 里氏替换 (Liskov Substitution)
- [ ] **I** — 接口隔离 (Interface Segregation)
- [ ] **D** — 依赖反转 (Dependency Inversion)

## 核心概念
- [ ] 封装 (Encapsulation)
- [ ] 继承 (Inheritance)
- [ ] 多态 (Polymorphism)
- [ ] 抽象 (Abstraction)

## 设计原则
- [ ] 组合优于继承
- [ ] 面向接口编程
- [ ] DRY / KISS / YAGNI
