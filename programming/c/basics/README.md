# C 语言

## 特点
- 编译型，接近硬件
- 手动内存管理 (malloc/free)
- 指针操作
- 所有现代操作系统内核语言

## 学习路线
- [ ] 基础: 变量/类型/运算符/控制流
- [ ] 函数与作用域
- [ ] 指针: 地址/解引用/指针运算
- [ ] 数组与字符串
- [ ] 结构体 (struct)
- [ ] 动态内存: malloc / calloc / realloc / free
- [ ] 文件 I/O: fopen / fread / fwrite
- [ ] 预处理器: #define / #include / 条件编译
- [ ] Makefile 编译

## 指针核心

| 表达式 | 含义 |
|--------|------|
| `int *p` | p 是指向 int 的指针 |
| `*p` | 解引用，取 p 指向的值 |
| `&x` | 取 x 的地址 |
| `p++` | 指针移动到下一个元素 |
