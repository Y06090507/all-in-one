# All-in-One 全能工具库

个人全能仓库，涵盖编程、设计、学习笔记和自动化工具。

## 目录结构

```
all-in-one/
├── programming/           # 编程语言
│   ├── python/
│   │   ├── patterns/      # 23种设计模式清单
│   │   ├── algorithms/    # 排序、搜索、动态规划
│   │   └── scripts/       # 实用脚本（项目模板等）
│   ├── javascript/
│   │   ├── patterns/      # JS 设计模式
│   │   ├── algorithms/    # JS 算法
│   │   └── snippets/      # 代码片段
│   └── data-structures/   # 数据结构知识
├── design/                # 软件设计
│   ├── oop/               # SOLID / 面向对象
│   ├── system-design/     # 系统设计案例
│   └── uml/               # UML 图
├── learning/              # 学习资源
│   ├── roadmaps/          # 学习路线图
│   ├── cheatsheets/       # 速查表
│   └── books/             # 书单笔记
├── scripts/               # 自动化脚本
├── configs/               # 配置文件
├── projects/              # 子项目
└── tools/                 # 工具集合
```

## 快速开始

```bash
git clone git@github.com:Y06090507/all-in-one.git
cd all-in-one
```
