# 学习路线图

## Python 全栈
1. 基础语法 → 数据类型 → 函数 → 类
2. 文件 I/O → 异常处理 → 装饰器 → 生成器
3. 数据库 (SQLite/PostgreSQL) → SQLAlchemy
4. Web: Flask → FastAPI → Django
5. 异步: asyncio → aiohttp
6. 测试: pytest → unittest → mock
7. 部署: Docker → Gunicorn → Nginx

## JavaScript / Node.js
1. ES6+ → 异步 (Promise/async-await)
2. Express → 中间件 → REST API
3. TypeScript → 类型系统
4. 测试: Jest → Vitest
